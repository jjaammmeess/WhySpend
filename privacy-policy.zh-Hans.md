## 我们是谁

本应用 **「花钱了」（MoodSpend）** 由独立开发者运营并提供。  
我们不会出售你的个人账单内容，亦不会将此类内容用于定向广告。

---

## 我们收集哪些信息

你主动提供或在使用功能时产生的数据，可能包括：

| 类别 | 示例 | 是否必需 |
|------|------|----------|
| **账单与情绪记录** | 金额、类型、分类、情绪标签、备注、消费时间 | 使用记账功能时 |
| **附件** | 你上传的小票或相关照片 | 可选 |
| **自定义选项** | 自定义分类、情绪标签等 | 可选 |
| **个人资料** | 昵称、头像（含预设或相册图片） | 可选 |
| **应用设置** | 语言、主题、货币、情绪提醒、复盘规则、iCloud 同步开关等 | 使用设置时 |
| **通知** | 应用内系统通知、消费提醒相关元数据 | 使用通知功能时 |
| **设备与诊断** | iOS 版本、应用版本；仅在崩溃或同步异常时可能产生的错误描述 | 自动、有限 |

> **我们不会** 要求你提供 Apple ID 密码、银行卡完整卡号或政府身份证件。

## 信息如何使用

我们处理信息的目的限于：

- 提供记账、情绪分析与复盘等核心功能  
- 在多台设备间同步你的数据（仅在你开启 **iCloud 同步** 时）  
- 根据你的设置发送情绪提醒与应用内通知  
- 支持数据导出、备份与恢复  
- 改进稳定性、修复错误与回应你的支持请求

## 存储、同步与保留

### 本机存储

默认情况下，数据保存在你的 **iPhone 本地**（SwiftData / 本地数据库）。

### iCloud 同步（可选）

若在 **我的 → 数据中心 → 云同步** 中开启同步：

- 数据将通过 **Apple iCloud / CloudKit** 在你登录的 **同一 Apple ID** 设备间同步  
- 同步由 **Apple** 提供基础设施与安全机制；我们不经由自有服务器中转你的账单内容  
- 关闭同步后，**新数据** 不再上传；已存在于 iCloud 的数据由 Apple 账户策略管理  

### 保留期限

- 数据在你设备或 iCloud 中保留，直至你 **删除记录**、**清除应用数据** 或 **删除应用**  
- 你可通过应用内 **JSON 备份** 自行留存副本

## 你的权利与选择

你可在应用内或通过邮件行使以下权利：

| 权利 | 操作方式 |
|------|----------|
| **访问与导出** | 数据中心 → 导出 JSON / CSV |
| **更正** | 编辑账单、资料与设置 |
| **删除** | 删除单条记录；或使用「清除本地数据」（请先备份） |
| **限制同步** | 关闭 iCloud 同步并重启应用 |
| **撤回同意** | 停止使用或删除应用；联系我们 |

我们将在合理期限内回复 [octboyu@gmail.com](mailto:octboyu@gmail.com) 的隐私相关请求。

## 信息安全

我们采取合理措施保护数据，包括：

- 依赖 iOS 与 Apple 提供的设备级加密能力  
- 通过 iCloud 时遵循 Apple CloudKit 的安全模型  
- 不在应用内明文存储 Apple 账户密码  

任何互联网传输都无法保证绝对安全；请妥善保管设备与 Apple ID。
