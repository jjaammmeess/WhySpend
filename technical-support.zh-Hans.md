## 快速入门

### 记一笔账

1. 点击底部导航栏中央的 **「+」** 按钮  
2. 填写金额、分类、情绪等信息后保存  
3. 在 **首页** 与 **账单** 中查看记录

### 主要页面

| 标签 | 功能 |
|------|------|
| 首页 | 今日概览、情绪光谱、最新账单 |
| 账单 | 完整账单列表与筛选 |
| 情绪复盘 | 图表与情绪热力环等分析 |
| 我的 | 设置、数据管理、帮助与支持 |

首次安装会展示 **引导页**；可在 **我的 → 帮助与支持 → 查看引导** 中再次观看。

## 设置说明

### 主题

**我的 → 基础设置 → 主题**

- **跟随系统**（默认）：随 iOS 浅色 / 深色模式切换  
- **浅色** / **深色**：固定外观

### iCloud 同步

**我的 → 高级功能 → 数据中心 → 云同步**

| 开关状态 | 说明 |
|----------|------|
| 开启 | 账单、设置与通知等可通过 iCloud 在已登录同一 Apple ID 的设备间同步 |
| 关闭 | 新数据仅保存在本机，不再上传至 iCloud |

**重要**：更改 iCloud 同步开关后，请 **完全退出应用并重新打开**（从多任务界面划掉后再启动），新的同步配置才会生效。

若显示「未登录 iCloud」，请前往 **系统设置 → Apple ID → iCloud** 登录并确保 iCloud 可用。

### 情绪提醒与复盘规则

- **情绪提醒**：**我的 → 高级功能 → 情绪提醒**  
- **复盘规则灵敏度**：**我的 → 高级功能 → 复盘规则**（与情绪提醒分开设置）

### 数据备份与导出

**我的 → 高级功能 → 数据中心**

- 支持 JSON 备份导出 / 导入、CSV 账单导出  
- 清除本地数据前请务必先备份

### 主货币

**我的 → 高级功能 → 主货币** 可锁定货币；未锁定时跟随系统区域设置。

## 常见问题（FAQ）

### 1. 换手机后数据不见了？

- 确认旧设备已开启 **iCloud 同步**，且使用 **同一 Apple ID**  
- 新设备登录 iCloud 后打开应用，首次启动可能需要等待云端导入（出现「正在恢复云端数据」时请保持网络畅通，也可按提示跳过）  
- 若曾关闭 iCloud 同步，数据可能仅存在于旧设备本地，请尝试从旧设备的 **JSON 备份** 恢复

### 2. 同步失败或一直显示同步异常？

1. 检查 Wi‑Fi / 蜂窝网络  
2. 确认 iCloud 账户状态正常（系统设置中无告警）  
3. 在 **云同步** 中查看错误提示  
4. 尝试关闭再开启 iCloud 同步，并 **重启应用**  
5. 仍无法解决请邮件联系支持，勿在邮件中包含敏感账单明细（可说明大致现象即可）

### 3. 主屏幕名称与预期不一致？

主屏幕名称由 **系统语言** 决定，与应用内「语言」设置相互独立。若需英文主屏幕名，请将 iPhone **系统语言** 设为英文（或添加英文为首选语言）。

### 4. 如何查看应用版本？

**我的 → 帮助与支持 → 关于应用 → 版本**

### 5. 隐私政策 / 使用条款在哪里？

**我的 → 帮助与支持 → 关于应用** 中提供入口（部分内容完善中）。

## 数据与隐私（摘要）

- 开启 iCloud 同步时，数据存储于 **你的私有 iCloud 账户**，由 Apple 提供同步与加密基础设施  
- 关闭 iCloud 同步时，数据主要保留在 **本机**  
- 我们不会通过本支持邮箱要求你提供 Apple ID 密码或完整财务明细

完整隐私政策发布后将通过应用内「关于应用」及本仓库链接提供。
